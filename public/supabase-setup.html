1 Open Supabase Dashboard

2 Run Database Migrations

  • Click SQL Editor in the sidebar
  • Click "New Query"
  • Open supabase/migrations/001_initial_schema.sql in your project
  • Copy ALL contents and paste into SQL Editor
  • Click "Run"
  • Wait for success message
  • Repeat for files 002 and 003

3 Configure Authentication

  • Click Authentication ‚Üí Settings
  • Uncheck "Enable email confirmations" (easier for testing)
  • Click Save

4 Add Redirect URL

  • Go to Authentication ‚Üí URL Configuration
  • Under "Redirect URLs", add: http://localhost:8080/reset-password
  • Click Save

5 Verify Setup

  • Click Table Editor - should see 4 tables
  • Click Storage - should see 2 buckets
